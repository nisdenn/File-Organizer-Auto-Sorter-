# 🔎 File Organizer (Auto Sorter)

A simple Python script to automatically organize files in a folder by file type.

## 🚀 Features
- Automatically sorts files into folders
- Supports:
  - Images
  - Videos
  - Documents
  - Music
  - Archives
- Creates folders automatically

## 📦 Tech Stack
- Python 3
- os, shutil (built-in modules)

## ▶️ How to Use

1. Run the script:
```bash
python main.py
````

2. Input your folder path:

```id="abc123"
C:/Users/YourName/Downloads
```

3. Files will be organized automatically 🎉

## 📁 Example Result

```id="xyz789"
Downloads/
│── images/
│── videos/
│── documents/
│── music/
│── archives/
│── others/
```

## ⚠️ Notes

* Files will be moved (not copied)
* Make sure to backup important files
